## 원서 읽기, 왜 중요한가?

| 방법 | 강점 |
|------|------|
| 영어 듣기 | 발음·리듬 습득 |
| 영어 말하기 | 즉각적 표현력 |
| **원서 읽기** | **어휘·문장 구조·논리 전개** |

원서는 격식체와 비격식체를 모두 담고 있어 업무 이메일부터 회의 발언까지 폭넓게 활용할 수 있는 표현을 자연스럽게 익힐 수 있습니다.

:::tip 시작하는 방법
처음에는 하루 **10~15페이지**부터 시작하세요.
모르는 단어를 모두 찾기보다 **문맥으로 유추**하는 연습이 더 효과적입니다.
:::
