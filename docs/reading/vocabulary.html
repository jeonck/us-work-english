어휘 카드 만들기​

단어 하나당 아래 네 가지를 기록하면 기억에 오래 남습니다.

항목예시
단어leverage
원문 문장"We need to leverage our existing network."
뜻~을 활용하다, 지렛대로 삼다
내 문장"Let's leverage the data we already have."

어휘 정리 도구​

도구특징
Anki간격 반복 알고리즘, PC·모바일 연동
Notion테이블 형식, 원문 링크 보관 쉬움
종이 노트손으로 쓰면 기억 효율 높음

직장에서 자주 쓰는 어휘 유형​

  • 동사구: circle back, touch base, drill down, pull together
  • 형용사: actionable, scalable, robust, streamlined
  • 명사: bandwidth, deliverable, stakeholder, roadmap

:::tip 어휘는 맥락으로 기억 단어만 외우지 말고 원문 문장째로 저장하세요. 맥락이 있으면 실제 사용 시 훨씬 자연스럽게 떠오릅니다. :::
