# 원서를 더 잘 읽는 전략

## 속독 vs 정독

| 방식 | 목적 | 언제 사용하나 |
|------|------|---------------|
| 속독 (Skimming) | 전체 흐름 파악 | 첫 회독, 긴 챕터 훑기 |
| 정독 (Close Reading) | 표현·구문 깊이 이해 | 인상적인 단락, 어려운 논증 |

## 모르는 단어 처리 원칙

1. **문맥으로 유추**: 앞뒤 문장에서 의미를 추정한다
2. **건너뛰기**: 줄거리에 영향이 없으면 그냥 넘긴다
3. **표시 후 나중에 확인**: 중요해 보이면 밑줄만 긋고 계속 읽는다
4. **즉시 사전 확인**: 반복적으로 나오고 의미가 중요한 단어만

## 읽기 루틴 만들기

```
하루 15분 루틴 예시

[읽기 전]  어제 읽은 내용 한 줄 요약 (2분)
[읽기 중]  집중 독서 (10분)
[읽기 후]  오늘 발견한 표현 1~2개 메모 (3분)
```

:::tip 오디오북 병행
같은 책의 오디오북을 구해 **눈으로 읽으면서 귀로 듣는** 방식을 시도해 보세요.
읽기 속도와 발음 감각을 동시에 훈련할 수 있습니다.
:::
