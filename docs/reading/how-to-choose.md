# 내 수준에 맞는 원서 고르기

## 수준별 기준

| 수준 | 기준 | 예시 장르 |
|------|------|-----------|
| 입문 | 페이지당 모르는 단어 5개 이하 | 어린이 챕터북, 쉬운 소설 |
| 중급 | 페이지당 모르는 단어 5~10개 | 대중 소설, 비즈니스 자기계발서 |
| 고급 | 전문 용어 포함, 복잡한 구문 | 전문 서적, 학술서, 고전 소설 |

## 직장인에게 맞는 장르

- **비즈니스·자기계발**: 실제 업무에 바로 쓸 수 있는 표현 습득
- **회고록·에세이**: 자연스러운 구어체와 문어체 혼합
- **논픽션**: 배경 지식과 전문 어휘 동시 확장

## 선택 체크리스트

- [ ] 첫 페이지를 읽었을 때 내용이 70% 이상 이해되는가?
- [ ] 주제가 흥미롭거나 업무와 관련 있는가?
- [ ] 오디오북 버전도 있어서 함께 들을 수 있는가?

:::tip Lexile 지수 활용
책 뒤편이나 출판사 사이트에서 Lexile 지수를 확인하세요.
**800–1000L** 수준이 일반 성인 원서의 평균 난이도입니다.
:::
