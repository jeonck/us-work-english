# "perched up high"

## 원문

> "Halmoni's house is perched up high,  
> so we have to run up a long set of stairs."

---

## "perch" — 높은 곳에 앉다/자리잡다

`perch` 원래 뜻: 새가 나뭇가지나 높은 곳에 **가볍게 앉는 것**.

그런데 집이나 건물에 쓰이면: **높은 곳에 위태롭게 또는 당당하게 자리잡은** 모습.

> 절벽 위의 성, 언덕 꼭대기의 집 — 이런 이미지에 `perched`를 씁니다.

---

## "perched up high" — 겹쳐서 강조

`perched` 자체도 이미 "높은 곳에"라는 뜻을 포함합니다. 여기에 `up high`까지 더해서 **"아주 높은 곳에 딱 올라앉아 있는"** 느낌을 강조합니다.

할머니 집은 `at the top of a hill` — 언덕 꼭대기에 새처럼 앉아 있습니다.

---

## 활용 예문

| 예문 | 뜻 |
|------|-----|
| "The cat is perched on the windowsill." | 고양이가 창틀에 앉아 있다 |
| "The village is perched on a cliff." | 마을이 절벽 위에 자리잡고 있다 |
| "She perched on the edge of the chair." | 그녀는 의자 끝에 살짝 걸터앉았다 |

---

## 왜 "perched"인가

`located` 나 `situated`도 "위치해 있다"는 뜻이지만 느낌이 다릅니다.

- `located` → 사무적, 중립적
- `perched` → **시각적, 생동감 있는** — 마치 새가 앉은 것처럼 아슬아슬하고 인상적인

할머니 집의 비밀스럽고 마법 같은 분위기와 딱 맞는 단어입니다.

:::tip 핵심 정리
**perch = (새처럼) 높은 곳에 자리잡다**

집, 마을, 건물이 높은 곳에 있을 때 생동감 있게 표현하는 단어.
:::
