# 슬랙 · 팀즈 소통

## 기본 원칙

- **짧고 명확하게** — 이메일처럼 길게 쓰지 않아도 됩니다
- **빠른 답장** — 24시간 이내가 기본 (급하면 @mention)
- **스레드 활용** — 채널이 복잡해지지 않도록

---

## 상황별 메시지

### 간단한 요청

```
Hey [Name], quick question — do you have the link to the design doc?

Hi, could you take a look at this PR when you get a chance? No rush!

@[Name] can you review this before EOD? Blocking me a bit.
```

### 상태 업데이트

```
Update: the deployment is done. Everything looks good ✅

Heads up — I'll be out this afternoon. [Name] can cover if anything urgent comes up.

FYI: I'm working on the bug fix. Should have a PR up by EOD.
```

### 잠깐 자리 비울 때

```
BRB (be right back) — grabbing lunch
AFK (away from keyboard) for ~30 min
Back online now
```

### 도움 요청할 때

```
I'm a bit stuck on [issue] — anyone dealt with this before?
Can someone help me debug this? I've been at it for a while 😅
[Name], do you have 10 min to pair on this?
```

## 자주 쓰는 슬랙 약어

| 약어 | 풀이 | 의미 |
|------|------|------|
| EOD | End of Day | 오늘 내로 |
| EOM | End of Message | 메시지 끝 (제목에 쓰면 본문 없음) |
| LGTM | Looks Good To Me | 승인 / 좋아 보여 |
| TLDR | Too Long, Didn't Read | 요약하면 |
| PR | Pull Request | 코드 리뷰 요청 |
| FYI | For Your Information | 참고로 |
| ASAP | As Soon As Possible | 최대한 빨리 |
| NVM | Never Mind | 됐어 / 신경 쓰지 마 |
