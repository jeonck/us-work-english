# 직장 소통 시작하기

취업 후 실제 업무에서 매일 마주치는 소통 상황을 다룹니다.

## 이 섹션에서 다루는 내용

| 주제 | 핵심 |
|------|------|
| [이메일](./email) | 업무 이메일의 톤과 구조 |
| [미팅](./meetings) | 회의 진행 표현과 참여하는 법 |
| [슬랙/팀즈](./slack-teams) | 메신저에서의 간결한 소통 |
| [스몰토크](./small-talk) | 자연스러운 일상 대화 |
| [발표](./presentations) | 프레젠테이션 핵심 표현 |

## 왜 직장 영어가 따로 필요한가?

시험 영어와 직장 영어는 다릅니다.

- **너무 formal한 표현** → 어색하게 들림 ("I would like to inquire about...")
- **너무 informal한 표현** → 비전문가처럼 들림 ("Hey, wanna check this out?")
- **직접적인 표현 vs 간접적인 표현** → 상황에 따라 다름

:::info 목표
자연스럽고 전문적인 **middle ground**를 찾는 것이 핵심입니다.
:::
