스몰토크

미국 직장에서 스몰토크는 관계 형성의 핵심입니다. 엘리베이터, 커피 머신, 회의 전 2~3분을 자연스럽게 채울 수 있어야 합니다.

대화 시작하기​

날씨 (가장 무난한 주제)​

Can you believe this weather? It's been so hot lately.
Finally some sunshine! How are you enjoying it?
I heard it's going to rain all week. Not great for my commute!

주말 / 휴일​

How was your weekend?
Do you have any fun plans this weekend?
Did you do anything special for the holiday?

최근 사건 / 회사 소식​

Did you catch the all-hands yesterday? What did you think?
I heard the new office opens next month — pretty exciting!
Have you tried that new lunch spot downstairs yet?

대화 이어가기​

Oh really? That sounds fun!
Nice! How did that go?
I've been meaning to try that. How was it?
Totally, I feel the same way.
That's so funny — same thing happened to me!

대화 자연스럽게 끝내기​

Anyway, I should get back to it. Good talking to you!
Well, I'll let you go. Have a great day!
Alright, I'll catch you later!
Nice chatting — talk soon!

피해야 할 주제 (미국 직장)​

  • 정치 (Politics) — 매우 민감
  • 종교 (Religion)
  • 연봉/돈 이야기
  • 외모 관련 발언

:::tip 잘 모를 때 쓰는 만능 표현 "That's interesting! Tell me more about that."

상대방이 뭘 말하든 대화를 자연스럽게 이어갈 수 있습니다. :::
