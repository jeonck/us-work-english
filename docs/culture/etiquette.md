# 미국 직장 예절

## 기본 원칙

### 1. 이름으로 부르세요
```
미국에서는 CEO도 이름으로 부릅니다.
"Mr. Smith" 보다 "Hey John" 이 훨씬 자연스럽습니다.
처음엔 어색해도 익숙해지면 편합니다.
```

### 2. 시간 약속 엄수
```
미팅 5분 전 준비 완료가 기본입니다.
늦으면 미리 알리세요: "Running 5 min late, sorry!"
```

### 3. 명확하게 말하기
```
"I'm not sure" → "I'm not sure, but I'll find out and get back to you."
"Maybe" → "I think we can, but let me confirm."
뭔가 모를 때 모른다고 하는 게 신뢰를 높입니다.
```

### 4. 적극적으로 참여하기
```
회의에서 아무 말도 안 하면 관심 없거나 이해 못한다고 생각합니다.
짧은 동의 한마디라도: "That makes sense." / "I agree with that."
```

---

## PTO (휴가) 문화

미국 직장의 휴가는 **당당하게 사용**하는 문화입니다.

```
요청할 때:
"I'd like to take PTO from [date] to [date]. I'll make sure everything is
covered before I leave."

승인 감사 표현:
"Thanks! I'll prepare a handoff doc so nothing falls through the cracks."
```

---

## 1:1 미팅 (One-on-one)

매니저와의 정기 미팅입니다. 적극적으로 활용하세요.

```
좋은 1:1 질문:
- "Is there anything I should be doing differently?"
- "What does success look like for me in the next quarter?"
- "What are the team's top priorities right now?"
- "Is there anything blocking the team that I can help with?"
```
