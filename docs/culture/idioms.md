# 자주 쓰는 비즈니스 이디엄

미국 직장에서 자주 들리는 표현들입니다. 알아듣기만 해도 큰 도움이 됩니다.

## 회의·업무 표현

| 표현 | 의미 | 예문 |
|------|------|------|
| **Hit the ground running** | 바로 성과를 내다 | "We need someone who can hit the ground running." |
| **Low-hanging fruit** | 쉽게 할 수 있는 것 | "Let's start with the low-hanging fruit." |
| **Bandwidth** | 여유/시간 | "Do you have the bandwidth to take this on?" |
| **On my radar** | 인지하고 있음 | "That's on my radar." |
| **Deep dive** | 깊이 분석하다 | "Let's do a deep dive into the data." |
| **Synergy** | 시너지 (협력 효과) | "There's real synergy between the two teams." |
| **Boil the ocean** | 불필요하게 크게 하다 | "Don't boil the ocean — just fix the core issue." |

## 진행·일정 표현

| 표현 | 의미 | 예문 |
|------|------|------|
| **Touch base** | 간단히 확인하다 | "Let's touch base on Friday." |
| **Circle back** | 나중에 다시 논의하다 | "Can we circle back on this tomorrow?" |
| **Move the needle** | 진전을 만들다 | "We need something that actually moves the needle." |
| **In the pipeline** | 진행 중인 | "We have several features in the pipeline." |
| **Drop the ball** | 실수하다 / 놓치다 | "I dropped the ball on that deadline, sorry." |
| **On the same page** | 같은 이해를 공유함 | "Are we all on the same page?" |

## 자주 쓰는 줄임 표현

| 표현 | 의미 |
|------|------|
| **EOD / COB** | End of Day / Close of Business (오늘 퇴근 전) |
| **OOO** | Out of Office (자리 비움) |
| **TBD** | To Be Determined (미정) |
| **WFH** | Work From Home (재택근무) |
| **RSVP** | 참석 여부 회신 부탁 |
| **Action item** | 해야 할 일 (할당된 업무) |
| **Deliverable** | 결과물 |
| **Stakeholder** | 이해관계자 |

:::tip 이디엄 자연스럽게 익히는 법
무조건 외우기보다 **들을 때마다 맥락과 함께** 기억하세요.
처음엔 알아듣는 것만으로도 충분합니다.
:::
