# 영어 말하기 시작하기

말하기는 가장 두려운 영역이지만, 가장 빠르게 는 영역이기도 합니다.

## 이 섹션에서 다루는 내용

| 주제 | 설명 |
|------|------|
| [발음](./pronunciation) | 원어민이 바로 알아듣는 핵심 발음 규칙 |
| [유창함](./fluency) | 막히지 않고 자연스럽게 말하는 법 |
| [필러워드](./fillers) | 생각할 시간을 버는 자연스러운 표현 |
| [연습법](./practice) | 혼자서도 할 수 있는 말하기 훈련 |

## 말하기 두려움 극복하기

많은 사람이 이런 생각으로 말을 못 합니다.

- "발음이 이상하면 어쩌지?"
- "문법이 틀리면 창피할 것 같아"
- "단어가 생각 안 나면?"

### 현실은?

```
미국 직장 동료들이 실제로 신경 쓰는 것:
✅ 당신이 말하려는 의미가 전달되는가
✅ 자신감 있게 말하는가

신경 쓰지 않는 것:
❌ 완벽한 발음
❌ 완벽한 문법
❌ 고급 어휘
```

:::info 핵심 마인드셋
**정확하게 말하는 것**보다 **용기 있게 말하는 것**이 먼저입니다.
틀려도 괜찮습니다 — 원어민도 말실수를 합니다.
:::
