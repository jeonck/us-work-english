# 말하기 연습법

말하기는 머릿속으로만 공부해서는 늘지 않습니다.
**입을 실제로 움직이는 연습**이 필요합니다.

---

## 혼자 할 수 있는 연습

### 1. 혼잣말 (Self-talk)

가장 쉽고 효과적인 방법입니다. 일상에서 영어로 생각하는 습관을 만듭니다.

```
아침에 일어나서:
"Alright, time to get up. What do I have today?
Morning standup at 10, then a 1:1 with my manager at 2.
I should prepare some updates."

코드 짜면서:
"Let me refactor this function. The issue is here...
I'll move this logic to a separate method."
```

### 2. 거울 말하기 (Mirror Practice)

거울을 보며 말하면 표정과 입 모양을 확인할 수 있습니다.

```
연습 주제:
- 오늘 한 일을 영어로 설명하기
- 주말에 무엇을 했는지 말하기
- 현재 진행 중인 프로젝트 설명하기
```

### 3. 녹음 후 듣기

스마트폰으로 자신의 목소리를 녹음하고 들으면 객관적으로 파악할 수 있습니다.

```
1. 주제를 정하고 2분간 말하기
2. 녹음 듣기
3. 개선점 파악 (발음, 속도, 필러 과다 등)
4. 다시 녹음
```

### 4. 쉐도잉

좋아하는 미드·팟캐스트·TED를 따라 말합니다.
([쉐도잉 훈련법 자세히 보기](/docs/listening/shadowing))

## 원어민과 대화 연습

### 언어 교환 (Language Exchange)

한국어를 배우려는 영어 원어민과 서로 가르치는 방식입니다.

| 플랫폼 | 특징 |
|--------|------|
| **Tandem** | 앱 기반, 텍스트·음성·영상 지원 |
| **HelloTalk** | 커뮤니티 기반, 교정 기능 있음 |
| **Speaky** | 무료, 다양한 언어 |

### AI 대화 연습

| 도구 | 활용법 |
|------|--------|
| **ChatGPT Voice** | 실제 대화처럼 말하고 피드백 받기 |
| **Claude** | 롤플레이 시나리오로 연습 |
| **Elsa Speak** | 발음 점수 측정 및 교정 |

```
AI에게 롤플레이 요청하는 법:
"Let's practice a mock job interview. You're the interviewer,
I'm the candidate applying for a software engineer position.
Please correct my grammar and pronunciation after each answer."
```

## 일주일 말하기 루틴 예시

| 요일 | 연습 |
|------|------|
| 월 | 혼잣말 10분 (오늘 할 일 영어로 말하기) |
| 화 | 쉐도잉 15분 |
| 수 | 거울 말하기 — 프로젝트 설명 연습 |
| 목 | AI 롤플레이 (회의 시나리오) |
| 금 | 녹음 후 피드백 |
| 토 | 언어 교환 파트너와 대화 |
| 일 | 미드 쉐도잉 or 자유 |

:::tip 완벽함보다 일관성
하루 1시간보다 **매일 15분**이 더 효과적입니다.
틀려도 계속 말하는 습관이 가장 중요합니다.
:::
