핵심 발음

완벽한 발음보다 알아듣기 쉬운 발음이 목표입니다. 이 몇 가지만 신경 써도 의사소통이 훨씬 명확해집니다.

한국인이 자주 틀리는 발음​

1. L vs R​

단어L 발음R 발음
light / right혀끝을 윗니 뒤에 댐혀를 어디에도 안 댐, 말아 올림
lead / read
collect / correct

연습:

L: "La La La" — 혀끝이 튕기는 느낌
R: "Rr Rr Rr" — 혀가 입 안에 떠 있는 느낌

2. V vs B​

단어V 발음B 발음
very / berry윗니가 아랫입술을 살짝 물음양 입술을 붙였다 뗌
vent / bent
자주 틀리는 단어:
very (V) ← 아주 중요한 V 발음
vest, vote, video, value

3. F vs P​

F: 윗니가 아랫입술을 건드림 — "ffff" 바람 소리
P: 입술을 붙였다가 터뜨림

file vs pile
feel vs peel
fast vs past

4. TH 발음​

혀끝을 윗니 사이에 살짝 넣고:
- 유성 th (the, this, that, there): 혀를 진동시킴
- 무성 th (think, thank, three): 바람만 내보냄

강세 (Stress)​

영어는 강세 언어입니다. 올바른 단어를 써도 강세가 틀리면 못 알아들을 수 있습니다.

명사 vs 동사 강세​

단어명사 (강세 앞)동사 (강세 뒤)
recordREcord (앨범)reCORD (녹음하다)
projectPROject (프로젝트)proJECT (투영하다)
permitPERmit (허가증)perMIT (허가하다)

자주 강세를 틀리는 단어​

develop    → de-VEL-op  (❌ DE-vel-op)
important  → im-POR-tant
understand → un-der-STAND
engineer   → en-gi-NEER

말하기 속도​

한국인은 종종 너무 빠르게 말하거나, 너무 조심스럽게 끊어 말합니다.

나쁜 예: "I...want...to...ask...about...the...deadline."
좋은 예: "I wanted to ask about the deadline."

자연스러운 속도 = 상대방이 편하게 들을 수 있는 속도

:::tip 발음보다 강세가 먼저 발음을 완벽하게 하려 하기 전에, 올바른 강세와 리듬을 먼저 익히세요. :::
