# 필러워드 — 생각할 시간 버는 표현

필러워드는 말하는 도중 생각을 정리할 때 쓰는 자연스러운 표현입니다.
원어민도 항상 씁니다. 없으면 오히려 어색하게 들립니다.

---

## 자연스러운 필러워드

| 표현 | 뉘앙스 |
|------|--------|
| **Well...** | 생각을 모을 때, 신중하게 |
| **So...** | 설명 시작할 때 |
| **I mean...** | 다시 말하거나 명확히 할 때 |
| **You know...** | 공감 유도, 캐주얼 |
| **Let me think...** | 잠깐 생각할 때 |
| **That's a good question.** | 시간 벌기 + 예의 |
| **Let me see...** | 기억하거나 확인할 때 |
| **Actually...** | 정정하거나 새 정보 추가할 때 |
| **Basically...** | 요약하거나 단순화할 때 |
| **To be honest...** | 솔직한 의견 말할 때 |

---

## 상황별 활용

### 갑자기 질문을 받았을 때

```
"That's a great question. Let me think about that for a second."
"Hmm, let me see... I think it's..."
"Well, there are a few factors here..."
"So, from my perspective..."
```

### 단어가 생각 안 날 때

```
"What's the word... it's like a... system that handles payments automatically."
"I'm blanking on the exact term, but basically it's a tool that..."
"You know what I mean — the thing that routes the requests?"
```

### 의견이 확실하지 않을 때

```
"I think... if I recall correctly..."
"I'm not 100% sure, but I believe..."
"Off the top of my head, I'd say..."
"I may be wrong, but..."
```

---

## 피해야 할 필러

| 피할 것 | 대신 쓸 것 |
|---------|-----------|
| "Uh... um..." (너무 많이) | Well / So / Let me think |
| 침묵 (너무 길게) | "Give me a second." |
| "How to say..." (번역투) | "What's the word..." |
| "It's difficult to explain..." | "Let me put it this way..." |

:::tip 자연스러운 필러의 양
필러가 **전혀 없으면** 어색하고 로봇 같습니다.
**너무 많으면** (um, uh 연발) 불안해 보입니다.
`Well`, `So`, `I mean` 정도를 자연스럽게 쓰는 게 이상적입니다.
:::
