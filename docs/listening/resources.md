## 유용한 유튜브 기능

### 속도 조절
처음엔 **0.75x**로 시작해서 익숙해지면 **1.25x**로 올리세요.
```
유튜브 설정(톱니바퀴) → 재생 속도
```

### 자동 자막
정확도가 완벽하진 않지만 모르는 단어 파악에 유용합니다.
```
유튜브 CC 버튼 → 영어 자막 켜기
```

## 하루 루틴 예시

```
출퇴근 (30분): 팟캐스트 듣기 (내용 이해 위주)
점심 (10분):   TED 클립 쉐도잉 1~2개
저녁 (15분):   미드 한 에피소드 (영어 자막)
```

:::tip 꾸준함이 핵심
하루 1시간보다 **매일 15분**이 훨씬 효과적입니다.
:::
