# 영어 듣기 시작하기

말하기보다 듣기가 먼저입니다. 상대방의 말을 정확히 알아들어야 대화가 가능합니다.

|------|------|

| [회의 듣기](./meetings) | 회의에서 놓치지 않고 따라가는 법 |
| [쉐도잉](./shadowing) | 듣기와 말하기를 동시에 훈련하는 법 |
| [추천 리소스](./resources) | 팟캐스트·유튜브·앱 추천 |

## 왜 미국 영어는 알아듣기 어려울까?

시험 영어와 실제 회화는 다릅니다.

| 시험 영어 | 실제 미국 영어 |
|-----------|--------------|
| 또렷하고 느린 발음 | 빠르고 연음이 많음 |
| 단어를 하나씩 발음 | 단어가 이어지고 줄어듦 |
| 문장이 완전함 | 축약·생략 많음 |

### 대표적인 연음 패턴

| 실제 발음 | 원래 표현 |
|-----------|-----------|
| "Whaddya" | What do you |
| "Gonna" | Going to |
| "Wanna" | Want to |
| "Kinda" | Kind of |
| "Didja" | Did you |
| "Lemme" | Let me |
| "Hafta" | Have to |

:::tip 핵심 조언
처음에는 100% 이해하려 하지 마세요.
**핵심 단어 30~40%만 들려도 맥락으로 충분히 파악**할 수 있습니다.
:::
