# 요청하는 표현

## 정중함 레벨별 표현

같은 요청도 상황에 따라 톤이 달라집니다.

### 급한 요청 (직접적)
```
Can you send me the file now?
I need this ASAP — sorry for the rush.
Could you prioritize this? It's blocking me.
```

### 일반 요청
```
Could you review this when you get a chance?
Would you be able to take a look at this?
Can I get your input on this?
```

### 공식적 / 부탁하는 느낌
```
Would it be possible to have this by Thursday?
I was wondering if you could help me with...
If you have a moment, could you...?
```

---

## 마감 관련 요청

| 상황 | 표현 |
|------|------|
| 마감 확인 | "What's the deadline for this?" |
| 마감 연장 요청 | "Is there any flexibility on the deadline?" |
| 우선순위 조율 | "I have a few things on my plate — what's most urgent?" |
| 도움 요청 | "I could use some help with this — do you have bandwidth?" |

---

## 미팅 요청

```
짧은 미팅:
"Do you have 15 minutes this week to chat about [topic]?"
"Can we jump on a quick call to sync on [project]?"

공식 미팅:
"Would you be available for a 30-minute meeting on [topic]?
I'll send a calendar invite with an agenda."
```

---

## 요청 거절하기

| 상황 | 표현 |
|------|------|
| 바빠서 못할 때 | "I'm a bit overloaded right now — can this wait until next week?" |
| 다른 사람 추천 | "I'm not the best person for this. [Name] would know more." |
| 일부만 가능 | "I can help with the first part, but I'll need more time for the rest." |
