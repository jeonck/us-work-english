# 표현 사전

상황별로 바로 꺼내 쓸 수 있는 표현 모음입니다.

## 구성

| 카테고리 | 내용 |
|----------|------|
| [일상 표현](./daily) | 매일 쓰는 기본 표현 |
| [요청하기](./requests) | 정중하고 자연스러운 요청 표현 |
| [의견 다르게 표현하기](./disagreeing) | 직접적이지만 예의 있게 |

## 사용법

각 표현은 **상황 → 한국어 의미 → 영어 표현** 순서로 정리되어 있습니다.
바로 복사해서 쓰거나, 상황에 맞게 살짝 바꿔 쓰세요.
