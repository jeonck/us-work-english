# 의견 다르게 표현하기

미국 직장에서 반대 의견은 **당연하고 환영받습니다.**
중요한 건 **어떻게** 표현하느냐입니다.

---

## 부드럽게 반대하기

```
I see your point, but I have a slightly different take on this.
That makes sense. I wonder though, have we considered...?
I'm not sure I fully agree — can I share a different perspective?
I think there might be another way to look at this.
```

## 데이터로 반대하기

```
The data seems to suggest something different.
Based on what I've seen, I think [alternative].
Looking at the numbers, I'm not sure that holds up.
The results from last quarter point in a different direction.
```

## 아이디어 개선 제안

```
I love the idea. What if we also considered...?
Great direction! One thing I'd add is...
This is solid. Could we take it a step further by...?
I think we're close — the only thing I'd tweak is...
```

## 강하게 반대해야 할 때

```
I have to respectfully disagree here.
I understand the reasoning, but I think this approach has some risks.
I'm concerned this might cause [problem] — can we talk through it?
I want to raise a concern before we move forward.
```

---

## 반대 후 합의 찾기

```
What if we combined both approaches?
Is there a middle ground here?
Could we pilot one approach first and see how it goes?
Let's take a vote / Let's see what the rest of the team thinks.
```

:::tip 기억하세요
반대 의견을 잘 표현하는 사람은 **신뢰받는 팀원**으로 인정받습니다.
침묵 = 동의로 오해받을 수 있습니다.
:::
