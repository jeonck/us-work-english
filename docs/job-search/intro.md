# 취업 영어 시작하기

미국에서 취업하려면 영어 실력만큼 **어떤 상황에서 무슨 말을 써야 하는지** 아는 것이 중요합니다.

## 이 섹션에서 다루는 내용

| 주제 | 설명 |
|------|------|
| [이력서 (Resume)](./resume) | 미국식 이력서 작성법과 핵심 표현 |
| [커버레터 (Cover Letter)](./cover-letter) | 지원 동기를 영어로 설득력 있게 쓰는 법 |
| [링크드인 (LinkedIn)](./linkedin) | 프로필 최적화와 네트워킹 메시지 |
| [인터뷰 (Interview)](./interview) | 자주 나오는 질문과 STAR 답변 구조 |

## 미국 취업의 현실

- **네트워킹**이 채용의 70% 이상을 차지합니다
- 이력서는 ATS(자동 필터링 시스템)를 통과해야 합니다
- 인터뷰에서는 구체적인 경험을 숫자로 증명하는 것이 핵심입니다

:::tip 시작하는 법
이력서 → 커버레터 → 링크드인 → 인터뷰 순서로 차근차근 준비하세요.
:::
